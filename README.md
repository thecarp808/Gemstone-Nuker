# Gemstone Nuker
_A QOL module for hypixel skyblock mining. For Fabric 26.1.2 or 26.2_ 

## Features 
- Gemstone ESP
- Auto route creator
- Clear route macro
- Gemstone macro
- Comm macro
- Gemstone Nuker

## Installation 
Download fabric api 26.1.2 or 26.2

### Steps 
1. Download the lastest version of the mod
2. Move the jar file into your mod folder `minecraft/mods`
3. Restart Minecraft

## Usage
_To open the GUI, the command is /gems!_ --- Enjoy!
